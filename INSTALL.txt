=head1 Linux
* Perl лучше обновить до версии 5.12.*

* Установить из репозитория пакет tesseract или tesseract-ocr (программа, которая будет распознавать капчу).
  Еще нужен imagemagick, для разгадывания капчи через tesseract и некоторых других штук.
  sudo apt-get install tesseract-ocr imagemagick

* Ставим нужные модули перла через cpan: 
  perl ./INSTALL.txt

  Можно ставить не через cpan, а установить уже собранные пакеты из репозитория
  (названия пакетов будут варироваться в зависимости от дистрибутива - ищите сами)
  Например для модуля Coro и AnyEvent на дебианоподобных дистрах:
  sudo apt-get install libcoro-perl libanyevent-perl

Поверялось на Ubuntu 12.04 LTS (perl 5.12.0) и Gentoo (perl 5.12.4)
